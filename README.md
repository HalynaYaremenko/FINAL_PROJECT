# 📊 E-commerce Data Analytics Project

Цей репозиторій містить технічну реалізацію фінального проєкту з аналізу даних електронної комерції.

Мета проєкту — дослідити динаміку продажів, поведінку клієнтів, ефективність знижок та провести базове моделювання для оцінки факторів впливу на дохід компанії.

Аналітичні висновки та інтерпретація результатів винесені в окремий файл **Report.md**.

### 🗂 Структура репозиторію

```text
FINAL_PROJECT/
│
├── .venv/               → Віртуальне середовище Python
├── Python/              → Jupyter Notebook з EDA та моделлю регресії
├── Screens/             → Зображення графіків і дашбордів
├── SQL/                 → SQL-запити для BigQuery
├── Tableau/             → Файл з графіками, аналітикою та дашбордами
│
├── project1_df.csv      → Датасет для аналізу
├── requirements.txt     → Список залежностей Python
├── Report.md            → Аналітичний звіт і висновки
├── presentation.html    → Презентація проєкту
└── README.md            → Опис структури проєкту
```

## 📁 Python/

Містить Jupyter Notebook з:

- Exploratory Data Analysis (EDA)
- Обробкою пропущених значень
- Побудовою лінійної регресії (Age → Gross/Net Amount)

#### Для запуску:

```bash
pip install -r requirements.txt
jupyter notebook
```

## 📁 SQL/

Містить SQL-запити для роботи з датасетом thelook_ecommerce у Google BigQuery.
Використано: JOIN, GROUP BY, HAVING, Агрегації

## 📁 Tableau/

Містить файл Tableau Public з усіма розрахунками, графіками та дашбордом.
Дашборд включає: динаміку доходів, аналіз ефективності знижок, сегментацію клієнтів, аналіз прибутковості міст по категоріях товарів.

## 📁 Screens/
Містить скріншоти візуалізацій, використаних у звіті та для презентації проєкту.

## 📄 Report.md
Основний аналітичний документ проєкту.
Містить:
- Постановку бізнес-задач
- Результати аналізу
- Інсайти
- Фінальні висновки
- Pекомендації
